# Breast Cancer Classification using Machine Learning

## Project Overview

The objective of this project is to compare the performance of three machine learning algorithms for breast cancer classification using the Breast Cancer Wisconsin dataset from scikit-learn.

The models evaluated in this project are:

- Decision Tree
- Random Forest
- Gradient Boosting

---

## Machine Learning Workflow

1. Problem Definition
2. Data Loading
3. Exploratory Data Analysis (EDA)
4. Data Preprocessing
5. Train/Validation/Test Split
6. Model Training
7. Model Evaluation
8. Model Comparison
9. Best Model Selection
10. Final Evaluation
11. Error Analysis

---

## Evaluation Metrics

The models were evaluated using:

- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix
- Classification Report

---

## Results

The comparison showed that the selected best model achieved the strongest predictive performance on both the validation and test datasets.

The model demonstrated excellent generalization ability with only a small number of misclassified samples.

---

## Error Analysis

Misclassified samples mainly occurred when malignant and benign tumors shared similar feature characteristics.

The confusion matrix indicated that the model produced very few false positives and false negatives.

---

## Limitations

- Single dataset
- Default hyperparameters
- No cross-validation
- Limited feature engineering

---

## Future Work

Future improvements include:

## Conclusion

This project successfully demonstrated an end-to-end machine learning workflow for breast cancer classification.

Among the three evaluated models, the selected best model achieved excellent classification performance and generalized well to unseen data.
